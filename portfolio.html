Spanner

Spanner is a website that allows users to analyse their listening habits and playlists. I undertook this project to learn about web development, gain some expereince on the front end, as well as giving me an escuse to build a sever to host the site! The frontend was developed in Typescript with React. The backend, which communicates with the Spotify API and processes the data, was developed in Go.

I only have access to a developer API token, which does not allow anyone to log onto the site. Instead, I have forked repository and created a second version that only displays my statistics on the website, using my account's API key.

Gesture Controlled Robot Hand

For my masters project, I proposed a gesture controlled robotic hand. I have designed and 3d printed a hand with 6 degrees of motion, being controlled via a microcontroller. The microcontroller recieves data from a Raspberry Pi, which position data via UDP from a desktop application over the internet. The desktop application tracks the user's hand using a webcam and the MediaPipe library.

Table Football Robot

At the Nottingham 24 hour Hackathon - HackNotts84, I led a team to Intel's 1st prize, building a table football robot and competing against over 40 teams from 25 Unis across the UK.

This was acheived using computer vision and parts of a 3D printer. We were able to track the location of the ball in real time using a webcam and OpenCV and send Gcode commands to the robot control board in order to intercept and kick the ball towards the goal.
